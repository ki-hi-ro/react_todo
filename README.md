# 資産化期の行動管理システム Frontend

## 人生を経営するためのタスク管理システム

Django REST Frameworkで構築したAPIと連携し、収入・技術資産・回復・戦略を管理するためのフロントエンドアプリケーションです。

日々の行動を見える化し、資産形成につながるタスクを管理することを目的としています。

---

## このプロジェクトの目的

このアプリは単なるToDoアプリではない。

**「人生を経営するためのタスク管理システム」**

として、

* 収入
* 技術資産
* 回復
* 戦略

の4つの視点から行動を管理し、日々の積み上げを資産形成につなげることを目的としている。

## 初期データ

| タスク                                | カテゴリ | 完了条件                  |
| ---------------------------------- | ---- | --------------------- |
| Renderのスリープ対策を調査し、解決方法を技術記事としてまとめる | 技術資産 | ブログ記事を1本公開する          |
| My Portfolioの制作物管理機能の詳細を記入する       | 技術資産 | READMEまたはポートフォリオに反映する |
| クラウドワークスのWordPress案件に3件応募する        | 収入   | 提案文を送信する              |
| 30分散歩する                            | 回復   | 散歩を実施する               |
| フェーズ3計画を30分かけて見直す                  | 戦略   | タスクと期限を更新する           |

## マイルストーン①「見える化」

### 目標

Django REST FrameworkのAPIからタスクを取得し、React画面へ表示する。

### 表示項目

* タスク名
* カテゴリ
* 完了状態
* 期限

### 完了基準

React画面に初期データが表示されること。
