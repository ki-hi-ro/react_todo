# 人生を経営するための行動管理システム

## 概要

Django REST Frameworkで構築したAPIと連携し、

* 営業
* 開発
* 健康
* 計画

の4つの視点から日々の行動を管理するためのフロントエンドアプリケーションです。

行動を見える化し、継続的な成長と価値創出につなげることを目的としています。

---

## システム構成

```text
React Frontend
    ↓
Django REST Framework API
    ↓
Database
```

## 開発状況

### マイルストーン①「見える化」 ✅ 完了

#### 目標

Django REST FrameworkのAPIからタスクを取得し、React画面へ表示する。

#### 表示項目

* タスク名
* カテゴリ
* 完了状態
* 期限

#### 完了基準

React画面に初期データが表示されること。

#### 実装内容

* Django REST Framework API連携
* タスク一覧取得
* カテゴリ表示
* 完了状態表示
* 期限表示
* TypeScript型定義追加

---

### マイルストーン②「登録」

#### 目標

React画面からタスクを登録できるようにする。

#### 完了基準

フォームから登録したタスクがデータベースへ保存されること。

#### 実装予定

* タスク登録フォーム
* POST API連携
* 登録後の自動再描画

#### 登録予定データ

| タスク                                | カテゴリ | 完了条件                  |
| ---------------------------------- | ---- | --------------------- |
| Renderのスリープ対策を調査し、解決方法を技術記事としてまとめる | 開発   | ブログ記事を1本公開する          |
| My Portfolioの制作物管理機能の詳細を記入する       | 開発   | READMEまたはポートフォリオに反映する |
| クラウドワークスのWordPress案件に3件応募する        | 営業   | 提案文を送信する              |
| 30分散歩する                            | 健康   | 散歩を実施する               |
| 今後の開発計画を30分かけて見直す                  | 計画   | タスクと期限を更新する           |

---

### マイルストーン③「更新」

#### 目標

* 状態更新
